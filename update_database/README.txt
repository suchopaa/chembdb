This is how to make an update, step by step:
1) Download the newest Chembl postgres dump.tar.gz to this folder
2) Download the newest all BindingDB data tsv.zip to this folder
3) Open file "run_1.sh" and write the names of files of databases there (without file types, so no ".dmp.tar.gz" or ".tsv.zip"
3) execute command "source run_1.sh"
4) Stop the server
5) execute command "source run_2.sh"
6) Launch the server
