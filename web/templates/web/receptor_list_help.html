Number of records and pages

In the top left corner, there is a number of all receptors. In case of 0 receptors, the information about the number of receptors is red.

Above and below the table, there is a current page number and a number of all pages. The maximum of receptors per page is 30.
In case of more pages, the buttons "first", "previous", "next", and "last" will appear.
