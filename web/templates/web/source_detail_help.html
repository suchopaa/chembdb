About the detail of source

In the detail of source, further information, such as patent number or authors, are listed.

This is what the detail of source should look like.

Individual values, numbers and characters will be described using the example above in the following table:

10.1016/j.bmcl.2008.03.027 The DOI of source, where the record originally come from. If the source is not available, there will be an "Unknown source" instead of the DOI. Click the DOI, the original source, where the activity come from, to open it.
18374567 The Pubmed ID. If the Pubmed ID is not available, there will be an "Unknown source" instead of Pubmed ID. Pubmed is a webside for the database MEDLINE. This database includes citations and abstracts of biomedical literature.
Unknown The Pubchem ID. If the Pubchem ID is not available, there will be an "Unknown source" instead of Pubchem ID. Pubchem is a database of many freely accessible chemical information, such as biological activities, patents, or citations.
None The number of patent (if exists, else "None").
Roth, J; Madoux, F; Hodder, P; Roush, WR The list of authors of the source (journal, article, etc.).
Scripps Florida The institution the source (journal, article, etc.) comes from.
