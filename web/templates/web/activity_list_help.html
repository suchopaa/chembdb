About the list of activities

This is the list of activities that works either as results of the search form or as a list of all activities.

This is what the list of activites should look like.
Depending on the user settings of the search form, the Tanimoto coefficient is present (either "0-1" or empty). In case of the list of all activities, Tanimoto is ommited.


Number of records and pages

In the top left corner, there is a number of all records. In case of 0 records, the information about the number of records is red.

Above and below the table, there is a current page number and a number of all pages. The maximum of records per page is 30.
In case of more pages, the buttons "first", "previous", "next", and "last" will appear.

Table of activities

Individual records, numbers, and characters are described using the example above in the following table:

Checkbox for the export selection of this record (more information about an export below). The checkbox in the header line will select every record of the current page.
Estrogen receptor The Uniprot name of receptor. To open the "Detail of receptor" page, click the name of the receptor.
The picture of ligand. To make the picture bigger, hover the cursor on the picture. To open the "Detail of ligand" page, click the picture.
1.0 The Tanimoto coefficient (if available). To get more information about the Tanimoto coefficient, see "About the search form" section.
Unknown source The DOI of source where the record originally come from. If the source is not available, there will be an "Unknown source" instead of the DOI. To open the "Detail of source" page, click the DOI of source.
IC50 The physical quantity of activity record.
= The relation (can be "<", ">", "=", etc.).
80.0 The value of activity record.
nM The unit (mostly nM).
Chembl The source database (or databases, in case the record is available in more than one).
Details Click the "Details" to get more information about the activity record, such as temperature or pH.
Click to sort the records according to the specific column. The first click will sort ascendingly (A-Z, 1-9), the second one descendingly (Z-A, 9-1).
Export
  • Export selected to .csv
    This button will export every checked (selected) record on the current page to .csv file.
  • Export all to .csv
    This button will export all records to .csv file.

The name of the export file follows given template: "export_data__YYYY_MM_DD__HH_MM_SS.csv".
The time is set according to the actual UTC time.
